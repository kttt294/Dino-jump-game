### Hướng Dẫn Khởi Động
1. Mở file `index.html` (trong thư mục `new-game`) bằng bất kỳ trình duyệt web nào.
2. Nhấn nút **"Bắt đầu ngay"**.
3. Sử dụng phím **Space (Cách)** hoặc phím **Mũi tên lên (Up Arrow)** để giúp chú Khủng Long (Dino) nhảy qua các rặng xương rồng gai góc. *Nếu bạn chơi trên điện thoại, chỉ cần chạm vào màn hình khung game.*
4. Tốc độ sẽ tăng dần! Hãy rèn luyện phản xạ để sống sót lâu nhất có thể và lập kỷ lục mới.

### Tính Năng
* **Giao diện Glassmorphism**: Thiết kế kính mờ hiện đại.
* **Vật lý mượt mà**: Trọng lực và lực nhảy được tinh chỉnh (nhảy nhanh, rơi nhanh) đòi hỏi sự tính toán chuẩn xác từ người chơi.
* **Lưu điểm cao**: Tích hợp tính năng tự động ghi nhớ Kỷ lục cao nhất (High Score) của bạn ngay trên trình duyệt (sử dụng `localStorage`).
* **Hoàn toàn bằng JavaScript thuần**: Tối giản, nhẹ nhàng và phản hồi tức thì mà không cần dùng đến các thư viện game nặng nề cồng kềnh.
